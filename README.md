# Airbnb Market Performance Analysis – Hawaii Region

This project analyzes Airbnb listing performance across Hawaii using pricing, revenue, reviews, and geographic demand patterns.

## Objectives

- Identify high-performing tourism regions
- Compare revenue across room types
- Visualize spatial demand clusters
- Build an interactive KPI-switching dashboard

## Tools Used

- Tableau
- Python (Google Colab)
- InsideAirbnb dataset

## Key Insights

- Honolulu generates the highest listing revenue
- Entire-home listings outperform other room types
- Coastal tourism zones show strong demand clusters
- Higher-rated listings show stronger pricing power
